# ADR-010

Title: Open Source First Strategy

Status: Accepted

Date: 2026-07-23

---

## Context

The Founder intends Genesis to remain accessible regardless of financial resources.

Open-source infrastructure promotes:

- transparency
- auditability
- portability
- community contribution
- independence

The Constitution identifies open-source operation as a foundational principle.

---

## Decision

Genesis shall adopt an Open Source First strategy.

Preference order:

1. Existing Open Source Solution
2. Self-Hosted Open Source Solution
3. Open Standard Implementation
4. Commercial Solution
5. Proprietary Dependency

Commercial services may be used when:

- required legally
- required operationally
- no practical alternative exists

However:

Genesis shall avoid unnecessary proprietary dependency whenever practical.

---

## Consequences

Benefits:

- Lower costs
- Greater transparency
- Community participation
- Organizational independence

Costs:

- Potentially higher operational complexity

This tradeoff aligns with Founder Intent.
