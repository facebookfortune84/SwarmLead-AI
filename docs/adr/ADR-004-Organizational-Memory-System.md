# ADR-004

Title: Organizational Memory System

Status: Accepted

Date: 2026-07-23

---

## Context

Critical knowledge often exists only:

- in conversations
- in individual memory
- in undocumented decisions

This creates fragility.

---

## Decision

Genesis shall implement an Organizational Memory System.

Components include:

- Founder Documents
- ADRs
- Governance Artifacts
- Knowledge Graph
- Documentation
- Project Intelligence Registry

Knowledge preservation becomes a core architectural responsibility.

---

## Consequences

Benefits:

- Continuity
- Discoverability
- Reduced rework

Costs:

- Maintenance effort

This tradeoff is accepted.
