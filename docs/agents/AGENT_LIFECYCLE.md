## Operational Review

Agents should be periodically evaluated for:

- Accuracy
- Value
- Governance Compliance
- Performance
- Knowledge Contribution

---

## Retirement

Retired agents remain documented.

History is preserved.

Capabilities are archived.

Knowledge is retained.

---

## Principle

Agent retirement should never destroy organizational memory.
