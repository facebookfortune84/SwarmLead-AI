## Purpose

The Agent Registry serves as the authoritative catalog of all agent roles operating within Genesis.

Every agent must have:

- Identity
- Purpose
- Responsibilities
- Permissions
- Escalation Path
- Accountability Chain

No agent may exist without registration.

No agent may operate anonymously.
