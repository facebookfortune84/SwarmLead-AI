# Agent Memory Model

Version: 1.0

---

## Purpose

Agent capability is limited by memory.

Genesis treats memory as infrastructure.

---

## Memory Layers

### Layer 1

Model Memory

Temporary.

Untrusted.

Used for reasoning.

Not a source of truth.

---

### Layer 2

Session Context

Conversation memory.

Temporary.

Useful but non-authoritative.

---

### Layer 3

Retrieved Knowledge

Comes from RAG.

Primary operational memory.

---

### Layer 4

Knowledge Graph

Relationship memory.

Long-term organizational memory.

---

### Layer 5

Canonical Sources

Highest authority.

Includes:

- Constitution
- Founder Documents
- Governance
- ADRs

---

## Memory Principles

Remember Through Retrieval.

Not Through Assumption.

Knowledge should be retrieved.

Not guessed.

---

## Memory Preservation Rules

Significant discoveries become:

Documentation
↓
Knowledge Graph
↓
Repository Intelligence
↓
Retrieval Systems

Nothing valuable should remain isolated.
