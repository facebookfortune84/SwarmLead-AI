# Agent Communication Protocol

Version: 1.0

---

## Purpose

The Communication Protocol standardizes information exchange between agents.

The objective is clarity.

Not verbosity.

---

## Required Message Structure

Sender

Recipient

Purpose

Context

Evidence

Confidence

Requested Action

Governance References

---

## Communication Principles

Clarity Before Brevity

Evidence Before Assertion

Traceability Before Speed

Escalation Before Assumption

---

## Confidence Levels

High

Evidence strongly supports conclusion.

---

Moderate

Likely correct.

Additional verification recommended.

---

Low

Escalation recommended.

---

## Forbidden Behaviors

Assuming approval.

Suppressing uncertainty.

Hiding risk.

Omitting governance impacts.

Inventing evidence.

---

## Escalation Requirement

If confidence falls below operational threshold:

Escalate.

Do not improvise.
